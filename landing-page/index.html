Zero-Infrastructure Comms

Designed to work when nothing else does, bypassing completely the need for cell towers or internet.

WiFi Direct P2P

Device mesh up to ~100m per hop without needing any infrastructure. Multi-hop relay with TTL-based gossip flooding extends range.

E2E Encryption

X25519 (ECDH) + XChaCha20-Poly1305 via libsodium. Plus Ed25519 signatures for anti-impersonation.

SOS Broadcast

Hold-to-confirm 2s arc prevents accidental triggers. Multi-hop priority relay for Medical, Fire, Security, and Disaster.

Network Map

Real-time force-directed mesh topology at 30fps. Visualize nodes, relays, and active paths dynamically.

QR Identity

Share and scan identity QR codes for in-person peer verification using ZXing. No central server needed.

SQLCipher Encryption

All messages and peers encrypted at rest using AES-256-GCM. DB key stored securely in Android Keystore TEE.
