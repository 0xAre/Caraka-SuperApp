### Efek Visual Penting
1. **Glitch effect** di opening → cari template "glitch transition" di CapCut
2. **Overlay teks** → background semi-transparan hitam, font bold putih
3. **Split screen** → tunjukkan 2 HP bersamaan saat relay terjadi
4. **Zoom in** ke network map → gunakan Ken Burns / slow zoom
5. **Color grading** → sedikit lebih gelap, tint amber/merah untuk dramatic

### Caption/Subtitle
Tambahkan subtitle Indonesia di bawah layar untuk semua narasi voice over.

## 📱 TIPS REKAM LAYAR HP

### Opsi 1 — Screen Record Built-in (paling mudah)
- Android: Swipe down → Screen Record
- Kualitas: 1080p jika available
- Audio: aktifkan "Record microphone" untuk suara notifikasi

### Opsi 2 — scrcpy (kualitas terbaik, via laptop)
```bash
# Install
sudo apt install scrcpy  # Linux
brew install scrcpy       # Mac

# Jalankan (sambungkan HP via USB)
scrcpy --record scene1_warga.mp4 --video-bit-rate 8M

# Multi-device
scrcpy --serial DEVICE_ID_1 --record hp1.mp4
scrcpy --serial DEVICE_ID_2 --record hp2.mp4
```

### Opsi 3 — Vysor / AirDroid (wireless)
- Install Vysor di laptop, pair dengan HP via USB
- Mirror layar HP ke laptop window
- Record window laptop dengan OBS
